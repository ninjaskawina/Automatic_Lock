enum hFramework::hFile::FileMode

File modes.

Enumerator
MODE_READ 

Specifies read access to the object. Data can be read from the file. Combine with MODE_WRITE for read-write access.

MODE_WRITE 

Specifies write access to the object. Data can be written to the file. Combine with MODE_READ for read-write access.

MODE_OPEN_EXISTING 

Opens the file. The function fails if the file is not existing.

MODE_OPEN_ALWAYS 

Opens the file if it is existing. If not, a new file is created. To append data to the file, use seek method after file open in this method.

MODE_CREATE_NEW 

Creates a new file. The function fails with hFile::ErrorCodes::ERROR_FILE_ERR if the file is existing.

MODE_CREATE_ALWAYS 

Creates a new file. If the file is existing, it will be truncated and overwritten.
