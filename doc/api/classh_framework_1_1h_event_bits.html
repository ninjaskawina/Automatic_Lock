Detailed Description

Manipulates eventBits structure.

One object can hold up to 24 bits (8 highest bits are discarded). Event bits are used to wait for one or multiple flags, without wasting CPU.

Returns
bits that are set in EventBits object. Before timeout return value should correspond to "bitMask".
hEventBits eb;
void hMain() {
while(1) {
eb.waitAll(0x0080AA01); // RTOS blocks current task until ALL bits: 0, 9, 11, 13, 15, 23 are set
// ... do some things
}
}
